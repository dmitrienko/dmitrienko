Технические задания

1

Перечислите известные вам инструменты и приёмы для отладки кода в разных браузерах.
Для отладки HTML, CSS и Javascript использую:
  • - Firefox: Firebug + FireCookie + Firefocus;
  • - Chrome: Firebug Lite, встроенный Developer Tools;
  • - Opera: Firebug Lite, Dragonfly;
  • - Safari: Drosera;
  • - IE: Developer Tools (F12).

2

Опишите своими словами, что происходит в браузерах при клике на ссылку yandex.ru (полнота ответа – на ваше усмотрение).
В общих чертах это происходит следующим образом:
  1. На ссылке yandex.ru генерируется событие клик;
  2. Если установлен обработчик события - он выполняется;
  3. Событие передается "вверх" по DOM-модели документа;
  4. Срабатывают любые обработчики, которые получили "всплывшее" событие;
  5. Когда событие доходит до корня модели документа, и если ни один из обработчиков не вернул false (или нет ни одного обработчика - простая ссылка на странице без скриптов), то происходет действие по умолчанию - переход по ссылке.

4

Найдите все текстовые файлы (*.txt), в имени которых содержится «yandex», а в содержимом — «школа фронтенд-разработки». (предполагается работа с командной строкой)
1-й вариант:
grep -e "школа фронтенд-разработки" *yandex*.txt
2-й вариант:
find . -name "*yandex*.txt" -exec grep "школа фронтенд-разработки" '{}' \; -print
